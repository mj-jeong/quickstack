## Setup Choices

- **pnpm** — [QuickStack opinion] Selected by the user during setup. Package manager used for all installs.
- **recommended preset** — [QuickStack opinion] Provides a structured src/ layout with common directories for scalable projects.
- **src/ directory + @/* import alias** — [QuickStack opinion] Separates source from config files; @/* alias reduces relative import paths.

## Directory Structure (QuickStack opinion)

- **src/components — shared UI components** — [QuickStack opinion] Centralizes reusable components to prevent duplication across features.
- **src/features — feature-scoped modules** — [QuickStack opinion] Groups related logic (components, hooks, utils) by domain for maintainability.
- **src/lib — utility libraries and clients** — [QuickStack opinion] Isolates third-party client setup (e.g., Supabase, API clients).
- **src/hooks — custom React hooks** — [QuickStack opinion] Keeps hooks separated and independently testable.
- **src/styles — global styles** — [QuickStack opinion] Clear home for global CSS and design tokens.
- **src/types — TypeScript type definitions** — [QuickStack opinion] Centralizes shared types to avoid circular dependencies.
